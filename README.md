# Fraud Detection Dashboard

## Overview
A web-based fraud detection dashboard that analyzes transaction data and visualizes fraudulent activities using Flask, MySQL, and Chart.js.

## Features
- View total accounts and transactions
- Detect fraudulent transactions
- Fraud vs Normal transaction visualization
- Category-wise transaction analysis
- Account-based transaction filtering
- Recent transaction monitoring

## Tech Stack
- Python
- Flask
- MySQL
- HTML
- CSS
- Bootstrap
- JavaScript
- Chart.js
